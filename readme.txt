== Description ==

A simple plugin allows you display related posts of post you are viewing

Shortcode [c4d-related-post image_size="thumbnail" cols="3" count="3"][/c4d-related-post]

<h2>List Params:</h2>
<ul>
<li><strong>image_size</strong>: thumbnail. Set image size of post.</li>
<li><strong>cols</strong>: The posts will be showed in row with number column you want.</li>
<li><strong>count</strong>: The number of post will be showed</li>
</ul>

To use full features, please install and active C4D Plugin Manager and Redux Framework

https://wordpress.org/plugins/c4d-plugin-manager/

https://wordpress.org/plugins/redux-framework/

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-related-post` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
